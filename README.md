# RaceDay — Календарь велогонок России 2026

**[raceday-russia.ru](https://raceday-russia.ru)** — агрегатор любительских велосипедных стартов России. Все гонки в одном месте: шоссе, гревел, MTB, велокросс, многодневки и байкпакинг.

## Возможности

- Календарь гонок, сгруппированный по месяцам
- Фильтрация по дисциплине, региону и месяцу
- Прямые ссылки на регистрацию у организаторов
- Адаптивный дизайн — работает на мобильных и десктопе

## Стек

| Слой | Технология |
|------|-----------|
| Frontend | HTML / CSS / Vanilla JS |
| Хостинг | GitHub Pages |
| Данные | Airtable REST API |
| Аналитика | Яндекс.Метрика |

Сайт полностью статический, без сборки и бэкенда. Данные загружаются из Airtable при открытии страницы.

## Данные

База гонок ведётся вручную. Перед регистрацией всегда проверяйте актуальность на сайте организатора — даты и условия могут измениться.

Если вы организатор и хотите добавить свою гонку — напишите на [raceday-russia.ru](https://raceday-russia.ru).
